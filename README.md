# nodeJs_Stydy

### 1. Requisitos

    1.1. NodeJs - > npm install nodejs
    1.2. MySQL  - > npm install mysql

### 2. Como executar

    2.1. Baixar e instalar dependências -> npm install
    2.2. Executar a aplicação -> npm start
    2.3. Executar testes automatizados - > npm test
